Gold (3+ build affixes) and Silver (2+) per gear slot — your keepers. Uniques, charms, greater affixes and item-power tiers each get their own color. Everything else hidden.

Round-trip verified

Every import code is decoded and validated before you copy it, so it imports cleanly into Diablo 4 the first time. Mythics are never touched — they drop with their natural beam.

How it works

  1. Download and run MedicK's Might.
  2. Click a top build, or paste a build URL / affix list.
  3. Tweak the options (live preview) and copy the import code.
  4. In Diablo 4: Inventory → Loot Filter → Import, paste.
